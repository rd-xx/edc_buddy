# edc_buddy

Étude de cas pour Buddy.

## Développement

### Prérequis

- [Flutter SDK](https://docs.flutter.dev/get-started/install)

### Installation

```bash
# Cloner le repository
$ git clone https://github.com/rd-xx/edc_buddy

# Se déplacer dans le dossier
$ cd edc_buddy

# Installer les dépendances
$ flutter pub get
```
